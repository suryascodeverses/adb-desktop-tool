# ADB Desktop Tool

A cross-platform desktop application for managing Android devices using ADB.

## Features
- Device detection
- APK metadata parsing
- Install / Uninstall / Launch apps
- Device install state sync
- (Planned) Logcat viewer

## Tech Stack
- Electron
- Next.js (App Router)
- TypeScript
- pnpm

## Development
```bash
pnpm install
pnpm --filter @adb/electron dev
```
